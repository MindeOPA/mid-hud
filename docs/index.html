mid-hud

A lightweight, customizable HUD & Speedometer for FiveM servers

View on GitHub Download ZIP Discord

Features

Status Rings

Health, armor, hunger, thirst, stamina, oxygen, stress with animated circular indicators

Speedometer

Real-time gauge with needle, tick marks, speed labels, RPM arc, and fuel arc

Smooth Animations

60fps rendering via requestAnimationFrame, no stuttering

Location Panel

Street name, area, compass heading display

Top Panel

Server ID, clock, cash, bank with animated money changes

Voice Indicator

3-level voice proximity display

Cinematic Mode

Hide HUD with keybind (default: Z)

Minimap Integration

Custom minimap positioning

Installation

1. Drop mid-hud into your resources folder
2. Add ensure mid-hud to your server.cfg
3. Configure config.lua to your liking
4. Restart server
